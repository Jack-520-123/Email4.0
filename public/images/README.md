# 背景图片说明

## 登录页面背景图片

请将您的背景图片重命名为 `login-background.jpg` 并放置在此目录下。

### 图片要求：
- 文件名：`login-background.jpg`
- 推荐尺寸：1920x1080 或更高分辨率
- 格式：JPG、PNG 均可（但文件名必须是 .jpg）
- 建议：选择色彩柔和、不会干扰登录表单阅读的图片

### 使用方法：
1. 将您的图片重命名为 `login-background.jpg`
2. 复制到 `public/images/` 目录下
3. 刷新登录页面即可看到效果

### 当前效果：
- 登录表单具有半透明背景和模糊效果
- 背景图片会自动适应屏幕大小
- 添加了半透明遮罩层确保文字清晰可读

如果您想使用不同的图片格式或文件名，请修改 `src/app/auth/login/page.tsx` 文件中的 `backgroundImage` 路径。
